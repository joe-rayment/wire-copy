Design Notes

- Overlay is centered both horizontally and vertically on screen

- Box width: ~60 chars (fits in 70-column terminals with margins)

- All box corners are rounded: ╭ ╮ ╰ ╯ (never sharp ┌ ┐ └ ┘)

- Category headers in green (PrimaryText) for visual grouping

- Key characters in cyan (AccentFg) — consistent with all key hint styling

- Action descriptions in medium green (SecondaryText) — dimmer, non-interactive

- Footer "Press ? or Esc to close" uses same cyan key / medium text pattern

- The overlay does NOT dim the background — it simply overwrites the center region and is cleared on dismiss (next ? or Esc press triggers full redraw)

- The help content is context-sensitive: it shows bindings for the CURRENT view

- Selection bar is not shown in the help overlay (it is not a navigable list)
