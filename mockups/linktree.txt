Assumes: 80x24 terminal, Phosphor theme (green-on-black)

COLOR KEY:
  [green]   = bright green #00ff00 (ANSI 46) — primary text
  [bright]  = slightly brighter green (ANSI 48) — content link titles
  [cyan]    = bright cyan #00ffff (ANSI 51) — interactive accent
  [dim]     = dark green #005f00 (ANSI 22-28) — borders, decorative
  [med]     = medium green #00af00 (ANSI 34) — secondary text, metadata
  [nav]     = muted green (ANSI 40) — navigation/external/footer links
  [white]   = white #ffffff (ANSI 15) — selected item text
  [sel-bg]  = dark green bg (ANSI 22) — selection background
  [hi-bg]   = green bg (ANSI 46) — search highlight background
  [hi-fg]   = black (ANSI 0) — search highlight foreground

================================================================================
INTERACTION NOTES
================================================================================

Navigation:
- j/k or ↓/↑: move between links
- h/l: collapse/expand sections; in grid, also moves columns
- Enter: open selected link in reader/browser
- s: save selected link to reading list
- A: save all links to reading list
- /: begin search (type query, press Enter)
- n/N: next/previous search match
- v: switch to reader view (if readable content available)
- R: force refresh page
- b: go back
- g/G: go to top/bottom

Search:
- Search highlights appear as [hi-bg][hi-fg] inverted blocks on matching text
- Status bar shows /query (n/N) when search is active
- Matches cycle through all visible and off-screen items

Cache indicators:
- Status bar shows progress: ▰▰▰▰▰▰▰▱▱▱ 17/24 cached
- When actively caching: amber-colored ▰ segment + current URL slug
- When complete: "all cached" text replaces progress bar
