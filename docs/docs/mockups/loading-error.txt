================================================================================
LOADING, ERROR, AND BOT CHALLENGE SCREENS
Assumes: 80x24 terminal, Phosphor theme (green-on-black)
================================================================================

COLOR KEY:
  [green]   = phosphor green #00d700 (ANSI 40) — primary text, action keys
  [title]   = playful pink #ff87d7 (ANSI 212) — emphasis, titles
  [cyan]    = bright cyan #00ffff (ANSI 51) — interactive accent
  [dim]     = dark green #005f00 (ANSI 22) — borders, decorative, structural
  [med]     = medium green #00af00 (ANSI 34) — secondary text
  [red]     = red #ff5f5f (ANSI 203) — error text
  [amber]   = amber #ffaf00 (ANSI 214) — warning/in-progress

================================================================================
DESIGN PRINCIPLES FOR THESE SCREENS
================================================================================

1. All centered, all boxed — consistent framing for "modal" states
2. Rounded corners ╭╮╰╯ on every box
3. Breathing room: blank lines inside boxes above/below content
4. Color coding:
   - [green] spinner = normal/automated operation
   - [amber] spinner = needs user attention
   - [red] headline = error state
5. Action hints always at bottom of box, keys in [cyan]
6. URLs always shown but never the brightest element
7. Box width: ~40 chars (fits in 50-column minimum terminal)
8. Vertical centering: box placed roughly 1/3 from top of terminal
