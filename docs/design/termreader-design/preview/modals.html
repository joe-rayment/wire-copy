Why this works · all chrome is text in cells, not CSS borders
 · ┌─┐ │ └─┘ thin · ╔═╗ ║ ╚═╝ double · ╭─╮ │ ╰─╯ round
 · Border color is just the foreground of those glyphs.
 · Width is fixed in cells (here 40), so corners always align.
 · Spinners are braille glyphs animated by frame, not CSS rotate.
