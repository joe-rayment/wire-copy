Why this works in a real terminal
 · Border is ╔═╗ box-drawing chars in column 0/W-1 of every row.
 · Color cycles by re-rendering border cells with a new fg color
   each tick — exactly what an ANSI palette swap does.
 · Marching chevrons shift one cell per tick. No CSS transforms.
 · Progress bar uses ▏▎▍▌▋▊▉█ for 8 sub-cell widths.
 · Spinner is one cell, 8 braille frames: ⠹⠸⠼⠴⠦⠧⠇⠏
