Rule · cyan keys are a dialect — never use cyan for anything else
 · Cyan #00ffff is reserved for keys. If a non-key element uses cyan,
   the dialect breaks and users lose the visual shortcut for "press this".
 · The label after a key is always med-green #00af00, never green.
 · Separator between hints: two spaces (status bar) or two-spaces·two-spaces.
