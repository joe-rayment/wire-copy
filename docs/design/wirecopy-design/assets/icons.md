# Unicode Icon Glossary (WireCopy)

All icons are Unicode glyphs. No emoji, no SVG, no icon font.

## Status / markers
- `●` U+25CF — unread
- `○` U+25CB — read
- `★` U+2605 — default collection
- `✔` U+2714 — success
- `✗` U+2717 — error / failure
- `⚡` U+26A1 — warning
- `↩` U+21A9 — undo
- `✦` U+2726 / `✧` U+2727 — celebration sparkles
- `♫` U+266B — podcast / audio

## Direction / tree
- `▶` U+25B6 — collapsed / play
- `▼` U+25BC — expanded / scroll-down
- `▲` U+25B2 — scroll-up
- `→` U+2192 — directional
- `←` U+2190 — back

## Selection / focus
- `▌` U+258C — left accent bar (selected row)
- `▎` U+258E — reader focus indicator (1/4 block)

## Metadata / layout
- `·` U+00B7 — metadata separator
- `…` U+2026 — truncation
- `—` U+2014 — em dash

## Box drawing
- `╭ ╮ ╰ ╯` U+256D/E/F, 2570 — rounded corners
- `─` U+2500, `│` U+2502, `┼` U+253C — lines / cross
- `╶──╴` — light item separator (U+2576, 2574)

## Progress / density
- `█ ▉ ▊ ▋ ▌ ▍ ▎ ▏` U+2588–F — eighth-block bar
- `░ ▒ ▓` U+2591–3 — decrypt noise
- `▰ ▱` U+25B0, 25B1 — cache progress

## Loading
- `⠋ ⠙ ⠹ ⠸ ⠼ ⠴ ⠦ ⠧ ⠇ ⠏` — braille spinner (80 ms/frame)
- `⠇` U+2847 — primary spinner glyph
