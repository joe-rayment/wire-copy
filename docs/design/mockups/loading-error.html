Design Notes
WHY: Esc:cancel prevents users from being trapped during slow Selenium fetches.
Retry success toast (Screen 2f) closes the error → recovery feedback loop.

All loading/error boxes use .d borders (structural chrome).
Error headlines use .r (red). Bot challenge and login spinners use .am (amber).
Normal loading spinners use .g (green). Details and URLs use .m.
Retry success checkmark uses .ok (success green #5fffaf).
