Assumes: 80x24 terminal, Phosphor theme (green-on-black)
================================================================================

  [cyan]    = bright cyan #00ffff (ANSI 51) — interactive accent, star
  [dim]     = dark green #005f00 (ANSI 22) — borders, structural
  [med]     = medium green #00af00 (ANSI 34) — counts, secondary
  [white]   = warm off-white #e4e4e4 (ANSI 254) — selected item text
  [sel-bg]  = dark green bg (ANSI 22) — selection background

================================================================================
INTERACTION NOTES
================================================================================

Navigation:
- j/k or ↓/↑: move selection
- h/l: move between columns (Variant B only)
- Enter: open selected collection (shows reading list)
- d: delete collection (confirmation prompt)
- :new <name>: create new collection
- :default: set selected as default collection
- b: go back to launcher
- ?: show help

Default collection:
- ★ marks the collection that receives items when you press 's' in link tree
- Changed via :default command or :set-default <name>
- Only one collection can be default at a time
