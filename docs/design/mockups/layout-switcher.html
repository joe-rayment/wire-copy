Phosphor 2.0 · AI Layout Preview System · Ctrl+L cycling toasts and status bar indicators
Layout Name Flash (500ms overlay on Ctrl+L)
╭───────────────────────╮ │ Layout: Dense List │ ╰───────────────────────╯
Centered overlay flashes for 500ms on Ctrl+L, then full screen redraws with the new layout applied and a status bar badge.
Layout name uses .b .bold for strong visual emphasis during the brief flash.

WHY: 500ms flash is brief enough to not feel like a mode switch but long enough to register. The layout name centered at large size creates a satisfying "click" moment.

First-use tip: First launch: Status bar shows "Tip: Ctrl+L to try different layouts" for the first session. Stored in UserSettingsStore; never shown again after first Ctrl+L press.

arstechnica.com [Layout: Dense List] │ all cached
When a non-default layout is active, the status bar line 2 shows the layout name between the domain and cache indicator. Badge uses .mu (#008787) — quiet, not distracting.
