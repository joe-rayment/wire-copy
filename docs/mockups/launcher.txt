================================================================================
LAUNCHER — Home Screen
Assumes: 80x24 terminal, Phosphor theme (green-on-black)
================================================================================

COLOR KEY:
  [green]   = phosphor green #00d700 (ANSI 40) — primary text, names
  [title]   = playful pink #ff87d7 (ANSI 212) — header titles, content links
  [cyan]    = bright cyan #00ffff (ANSI 51) — interactive accents, badges
  [dim]     = dark green #005f00 (ANSI 22) — borders, decorative, structural
  [med]     = medium green #00af00 (ANSI 34) — secondary text, domains
  [white]   = warm off-white #e4e4e4 (ANSI 254) — selected item text
  [sel-bg]  = dark green bg (ANSI 22) — selected item background
  [pink]    = vivid pink #ff5fd7 (ANSI 206) — celebration accent (rare)

================================================================================
INTERACTION NOTES
================================================================================

Navigation:
- j/k or ↓/↑: move selection between cards
- h/l or ←/→: move between columns (grid variants)
- 1-9: jump directly to bookmark by badge number
- o: focus URL bar
- Enter: open selected bookmark/URL
- a: add new bookmark (prompts for URL)
- d: delete selected bookmark (confirmation prompt)
- c: go to reading list / collections
- :: enter command mode
- ?: show help overlay

Selection behavior:
- Current selection gets ▌ accent bar on left edge
- Full row in cell gets [sel-bg] background
- Text switches to [white] foreground
- All other cards remain [green] text, no background

Layout switching:
- Ctrl+L or :layout cycles A→B→C→A
- Toast appears briefly: "Layout: Grid 2-col (A)" / "Layout: List (B)" / etc.
- Preference persists via UserSettingsStore
