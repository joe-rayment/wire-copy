================================================================================
ANIMATIONS — Storyboards
Assumes: 80x24 terminal, Phosphor theme (green-on-black)
================================================================================

## Architectural Constraint
All animations run SYNCHRONOUSLY during the render cycle. The app has no
background render thread. Animations block user input for their duration
(max 2 seconds). Implementation uses Thread.Sleep() between Console.Write()
calls with cursor repositioning. Only targeted screen regions are updated
per frame — no full-screen redraws during animation.

Terminal animations are frame-based — each "frame" is a targeted cursor update.
Frame rate target: ~10-15 fps (66-100ms per frame) for smooth feel without
excessive CPU usage.

COLOR KEY:
  [green]   = phosphor green #00d700 (ANSI 40) — primary text
  [title]   = playful pink #ff87d7 (ANSI 212) — emphasis, titles
  [cyan]    = bright cyan #00ffff (ANSI 51) — accent
  [dim]     = dark green #005f00 (ANSI 22) — decorative, structural
  [med]     = medium green #00af00 (ANSI 34) — secondary
  [pink]    = vivid pink #ff5fd7 (ANSI 206) — celebration
  [amber]   = amber #ffaf00 (ANSI 214) — progress, warning

NOTES:
- Single braille character rotates in place
- "Loading..." text is static
