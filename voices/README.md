# voices/ — local narration voice samples

Drop a **~10-second, clean, single-speaker** audio clip here (wav / mp3 / flac /
m4a / ogg) and pick it in Settings (`c`) → **Voice sample** to have the local
Chatterbox narration engine clone that voice.

- A quiet recording of natural speech clones best; avoid music or crosstalk.
- No sample set → Chatterbox uses its built-in voice.
- Everything in this folder **except this README is gitignored** — your samples
  never get committed.
